BAS

Building Automation Systems BAS such as heating, cooling, security, access control.

CIM

Computer Integrated Manufacturing CIM is a combination of different technologies and systems such as robotics, manufacturing

Data Historians

Data Historians store process data and perform contextual analysis.

DCS

Distributed Control Systems DCS are used in manufacturing and process control.

ERP

Enterprise Resource Planning ERP systems help organizations automate business processes. Plant production schedules, material use, shipping, inventory levels can be streamlined and optimized with ERP.

HMI

Human-machine interfaces (HMI) connect to DCS and PLCs to allow field operators basic controls and monitoring.

ICS

Industrial Control Systems (ICS) are hardware/software that run the local process.

MES

Manufacturing Execution Systems (MES) collect real-time data to help optimize production.

MOM

Manufacturing Operations Management (MOM) systems are software designed to optimize the manufacturing process.

RTU

Remote Terminal Units (RTU) connect sensos, actuators and other machinery to SCADA and DCS systems.

SCADA

Supervisory Control and Data Acquisition (SCADA) are systems that capture data and can be used to control systems at remote locations.

SIS

Safety Instrumented Systems SIS for ICS or SCADA using threshold values as an example.
